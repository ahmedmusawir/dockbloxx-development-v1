- `page.tsx` — **Server Component** (no `"use client"`, uses `async`/`await` and `generateStaticParams`).
- `DealerPageContent.tsx` — **Server Component** (no `"use client"` directive).
- `DealerCouponClientBlock.tsx` — **Client Component** (`"use client"` on line 1).

### d. How the `coupon` query param is read

In `DealerCouponClientBlock.tsx` lines 20–21:

```tsx
const searchParams = useSearchParams();
const couponCode = searchParams.get("coupon") || "";
```

Read client-side via `next/navigation`'s `useSearchParams`. The `dealerSlug` path segment is read separately, server-side, in `page.tsx` via `props.params.dealerSlug` and used only to fetch dealer-page CMS data (`fetchDealerPageData`) — it is not joined with the coupon code.

### e. What it does with the coupon today

In `DealerCouponClientBlock.tsx` `useEffect` (lines 40–55):

1. Calls `fetchCouponByCode(couponCode)` (REST round-trip to `/api/get-coupon-by-code`).
2. If found, calls `applyCouponToStore(coupon)` — the Zustand `applyCoupon` action — which immediately persists the coupon to `localStorage` via the `checkout-storage` persist middleware.
3. Sets local `appliedCoupon` state and renders the "Coupon Applied: {code}" UI plus a copy-to-clipboard button.

There is no signed token, no httpOnly cookie, no server-side dealer-attestation step. The flow is purely client-side, with the coupon object pushed into the same `useCheckoutStore` that the checkout page reads.

const NEXT_API_URL = process.env.NEXT_PUBLIC_APP_URL;

export const fetchCouponByCode = async (
  couponCode: string
): Promise<Coupon | null> => {
  try {
    if (!NEXT_API_URL) {
      console.error("Missing NEXT_PUBLIC_APP_URL in environment variables");
      return null;
    }

    const response = await fetch(
      `${NEXT_API_URL}/api/get-coupon-by-code?code=${couponCode}`
    );

    if (!response.ok) {
      console.warn(`Coupon not found or invalid: ${couponCode}`);
      return null;
    }

    const rawCoupon = await response.json();
    const coupon = rawCoupon[0];

    if (!coupon) return null;

    return {
      id: coupon.id,
      code: coupon.code,
      description: coupon.description,
      discount_type: coupon.discount_type,
      discount_value: parseFloat(coupon.amount),
      free_shipping: coupon.free_shipping,
      min_spend: coupon.minimum_amount,
      max_spend: coupon.maximum_amount,
      products_included: coupon.product_ids || [],
      products_excluded: coupon.excluded_product_ids || [],
      categories_included: coupon.product_categories || [],
      categories_excluded: coupon.excluded_product_categories || [],
      usage_limit: coupon.usage_limit,
      usage_count: coupon.usage_count,
      usage_limit_per_user: coupon.usage_limit_per_user,
      used_by: coupon.used_by || [],
      expires_on: coupon.date_expires,
      meta_data: coupon.meta_data,
    };
  } catch (error) {
    console.error("Error fetching coupon:", error);
    return null;
  }
};

import { WC_REST_URL } from "@/constants/apiEndpoints";
import { NextResponse } from "next/server";

const BASE_URL = WC_REST_URL;
const CONSUMER_KEY = process.env.WOOCOM_CONSUMER_KEY;
const CONSUMER_SECRET = process.env.WOOCOM_CONSUMER_SECRET;

export async function GET(req: Request) {
  try {
    const { searchParams } = new URL(req.url);
    const couponCode = searchParams.get("code");

    if (!couponCode) {
      return NextResponse.json(
        { error: "Coupon code is required" },
        { status: 400 }
      );
    }

    const url = `${BASE_URL}/coupons?code=${couponCode}&consumer_key=${CONSUMER_KEY}&consumer_secret=${CONSUMER_SECRET}`;

    const response = await fetch(url);
    if (!response.ok) {
      return NextResponse.json({ error: "Coupon not found" }, { status: 404 });
    }

    const data = await response.json();
    return NextResponse.json(data);
  } catch (error) {
    console.error("Error fetching coupon:", error);
    return NextResponse.json(
      { error: "Internal Server Error" },
      { status: 500 }
    );
  }
}

export function parseCouponMeta(coupon: Coupon): CouponMeta {
  const meta: CouponMeta = {};

  if (!Array.isArray(coupon.meta_data)) {
    return meta;
  }

  coupon.meta_data.forEach(({ key, value }) => {
    if (key === "_dockbloxx_discount_percent_per_product") {
      const percentValue = Number(value);
      if (!isNaN(percentValue)) {
        meta.percentPerProduct = percentValue;
      }
    }

    if (key === "_dockbloxx_allowed_emails") {
      const emails = Array.isArray(value)
        ? value
        : String(value)
            .split(",")
            .map((v) => v.trim());

      meta.allowedEmails = emails
        .filter((email) => email)
        .map((email) => email.toLowerCase());
    }

    if (key === "_expiry_time") {
      meta.expiryTime = String(value).trim();
    }

    if (key === "_expiry_timezone") {
      const rawTimezone = String(value).trim();
      const match = rawTimezone.match(/\]\s*(.+)$/);
      meta.expiryTimezone = match ? match[1].trim() : rawTimezone;
    }
  });

  return meta;
}

export function isCouponExpiredByTimezone(
  coupon: Coupon,
  meta: CouponMeta
): boolean {
  if (!coupon.expires_on) {
    return false;
  }

  if (!meta.expiryTime || !meta.expiryTimezone) {
    console.warn(
      `[Coupon ${coupon.code}] Missing timezone data. Falling back to date-only check.`
    );
    return new Date() > new Date(coupon.expires_on);
  }

  try {
    const expiryDate = coupon.expires_on;
    const timezone = meta.expiryTimezone;

    const nowInTZ = new Date().toLocaleDateString('en-CA', {
      timeZone: timezone,
      year: 'numeric',
      month: '2-digit',
      day: '2-digit'
    });

    const isExpired = nowInTZ > expiryDate;

    return isExpired;
  } catch (error) {
    console.error(`[Coupon ${coupon.code}] Error parsing timezone data:`, error);
    return new Date() > new Date(coupon.expires_on);
  }
}

export const validateCoupon = (
  coupon: Coupon,
  checkoutData: CheckoutData
): { isValid: boolean; message: string } => {
  const meta = parseCouponMeta(coupon);
  const userEmail = checkoutData.billing.email?.trim().toLowerCase();
  const userZipCode = checkoutData.shipping.postcode?.trim();

  // 0. REQUIRE EMAIL (Security Check)
  if (!userEmail) {
    return {
      isValid: false,
      message: "Please enter your email address to apply a coupon.",
    };
  }

  // 0.1 REQUIRE ZIP CODE (Shipping Check)
  if (!userZipCode) {
    return {
      isValid: false,
      message: "Please enter your shipping zip code to apply a coupon.",
    };
  }

  // 1. Email Restriction Check
  if (meta.allowedEmails && meta.allowedEmails.length > 0) {
    const isMatch = meta.allowedEmails.includes(userEmail);
    if (!isMatch) {
      return {
        isValid: false,
        message: "This coupon is restricted to specific users.",
      };
    }
  }

  // 2. 100% "FREE" coupon — total quantity-of-eligible-items check
  if (meta.percentPerProduct === 100) {
    const eligibleItems = checkoutData.cartItems.filter((item) =>
      coupon.products_included.includes(item.id)
    );
    const totalEligibleQuantity = eligibleItems.reduce(
      (sum, item) => sum + item.quantity,
      0
    );
    if (totalEligibleQuantity > 1) {
      return {
        isValid: false,
        message: "Free item coupons are limited to a total quantity of one.",
      };
    }
  }

  // 2.2 Expiration Check (Timezone-Aware)
  if (isCouponExpiredByTimezone(coupon, meta)) {
    return { isValid: false, message: "This coupon has expired." };
  }

  // 3. Min/max spend
  const subtotal = checkoutData.subtotal;
  const minSpend = parseFloat(coupon.min_spend);
  const maxSpend = parseFloat(coupon.max_spend);

  if (minSpend > 0 && subtotal < minSpend) {
    return {
      isValid: false,
      message: `Your order must be at least $${minSpend.toFixed(2)} to use this coupon.`,
    };
  }
  if (maxSpend > 0 && subtotal > maxSpend) {
    return {
      isValid: false,
      message: `This coupon can only be used on orders up to $${maxSpend.toFixed(2)}.`,
    };
  }

  // 4. Product / category include / exclude
  const cartProductIds = checkoutData.cartItems.map((item) => item.id);
  const cartCategoryIds = checkoutData.cartItems.flatMap((item) => item.categories);
  const cartCategoryIdsOnly = cartCategoryIds.map((category) => category.id);

  if (
    coupon.products_included.length > 0 &&
    !cartProductIds.some((id) => coupon.products_included.includes(id))
  ) {
    return {
      isValid: false,
      message: "This coupon is not valid for any items in your cart.",
    };
  }

  if (
    coupon.products_excluded.length > 0 &&
    cartProductIds.some((id) => coupon.products_excluded.includes(id))
  ) {
    return {
      isValid: false,
      message: "This coupon cannot be used with some items in your cart.",
    };
  }

  if (
    coupon.categories_included.length > 0 &&
    !cartCategoryIdsOnly.some((id) => coupon.categories_included.includes(id))
  ) {
    return {
      isValid: false,
      message: "This coupon is not valid for your selected product categories.",
    };
  }

  if (
    coupon.categories_excluded.length > 0 &&
    cartCategoryIdsOnly.some((id) => coupon.categories_excluded.includes(id))
  ) {
    return {
      isValid: false,
      message: "This coupon cannot be used with some categories in your cart.",
    };
  }

  // 5. Global usage limit
  if (
    coupon.usage_count &&
    coupon.usage_limit &&
    coupon?.usage_count >= coupon.usage_limit
  ) {
    return {
      isValid: false,
      message: "This coupon has reached its maximum usage limit.",
    };
  }

  // 6. Per-user usage limit
  const userUsageCount = coupon.used_by.filter(
    (email) => email.toLowerCase() === userEmail
  ).length;

  if (
    coupon.usage_limit_per_user &&
    userUsageCount >= coupon.usage_limit_per_user
  ) {
    return {
      isValid: false,
      message: `You have already used this coupon the maximum number of times (${coupon.usage_limit_per_user}).`,
    };
  }

  return { isValid: true, message: "" };
};

export const applyCoupon = (
  coupon: Coupon,
  checkoutData: CheckoutData
): CheckoutData => {
  let discountAmount = 0;
  let updatedShippingCost = checkoutData.shippingCost;

  switch (coupon.discount_type) {
    case "fixed_cart":
      discountAmount = coupon.discount_value;
      break;
    case "percent":
      checkoutData.cartItems.forEach((item) => {
        const itemLineTotal = item.price * item.quantity;
        let itemDiscount = (itemLineTotal * coupon.discount_value) / 100;
        itemDiscount = parseFloat(itemDiscount.toFixed(2));
        discountAmount += itemDiscount;
      });
      break;
    case "fixed_product":
      discountAmount = checkoutData.cartItems.reduce((total, item) => {
        if (coupon.products_included.includes(item.id)) {
          return total + (item.price * item.quantity * coupon.discount_value) / 100;
        }
        return total;
      }, 0);
      break;
  }

  discountAmount = Math.min(discountAmount, checkoutData.subtotal);

  if (coupon.free_shipping) {
    updatedShippingCost = 0;
  }

  return {
    ...checkoutData,
    discountTotal: discountAmount,
    total: checkoutData.subtotal + updatedShippingCost - discountAmount,
    shippingCost: updatedShippingCost,
    coupon: {
      ...coupon,
      code: coupon.code,
      description: coupon.description,
      discount_value: discountAmount,
      free_shipping: coupon.free_shipping,
    },
  };
};

export const removeCoupon = (checkoutData: CheckoutData): CheckoutData => {
  return {
    ...checkoutData,
    discountTotal: 0,
    total: checkoutData.subtotal + checkoutData.shippingCost,
    coupon: null,
  };
};

export function calculateCouponDiscount(
  coupon: Coupon,
  cartItems: CartItem[],
  subtotal: number
): number {
  const meta = parseCouponMeta(coupon);

  if (isCouponExpiredByTimezone(coupon, meta)) {
    return 0;
  }

  const minSpend = parseFloat(coupon.min_spend || "0");
  const maxSpend = parseFloat(coupon.max_spend || "0");
  if (subtotal < minSpend) return 0;
  if (maxSpend > 0 && subtotal > maxSpend) return 0;

  let discount = 0;
  switch (coupon.discount_type) {
    case "fixed_cart":
      discount = Number(coupon.discount_value);
      break;
    case "percent":
      discount = (Number(coupon.discount_value) / 100) * subtotal;
      break;
    case "fixed_product":
      discount = cartItems.reduce((acc, item) => {
        if (coupon.products_included.includes(item.id)) {
          return acc + Number(coupon.discount_value) * item.quantity;
        }
        return acc;
      }, 0);
      break;
  }

  return Math.min(discount, subtotal);
}

export const getCouponsFromStorage = (): Coupon[] => {
  const script = document.getElementById("coupon-data");
  if (!script) {
    return [];
  }

  try {
    const coupons: Coupon[] = JSON.parse(script.textContent || "[]");
    return coupons;
  } catch (error) {
    console.error("Error parsing coupon data:", error);
    return [];
  }
};

- **Address/shipping check reads:** `checkoutData.shipping.postcode` (trimmed) — `couponUtils.ts:263`. This is a presence check only (rule 0.1); zip value itself is not matched against any allow-list.
- **Other rules:** expiry (timezone-aware), min/max spend, products_included/excluded, categories_included/excluded, global `usage_count >= usage_limit`, per-user `used_by` count vs `usage_limit_per_user`, and a 100%-free quantity cap of 1 per cart.
- **`individual_use`:** not present in the validator.

### d. `meta_data` keys read

- `_dockbloxx_discount_percent_per_product` → `meta.percentPerProduct`
- `_dockbloxx_allowed_emails` → `meta.allowedEmails` (accepts array or comma-separated string; lowercased)
- `_expiry_time` → `meta.expiryTime` (HH:MM)
- `_expiry_timezone` → `meta.expiryTimezone` (IANA, with `[UTC±NN:NN] Region/City` prefix stripped)

### c. Where in the checkout flow this happens

Two distinct entry points feed the same Zustand `applyCoupon` action:

1. **Dealer landing page** (`/dealer-coupon/[dealerSlug]/?coupon=…`): auto-fires on mount via `useEffect` in `DealerCouponClientBlock`. No user interaction. The user has not yet entered email/zip — at this point `checkoutData.billing.email` and `checkoutData.shipping.postcode` are empty strings (defaults from store init), so `validateCoupon` rules 0 and 0.1 fail and the coupon **silently does NOT attach** (the store logs `"Invalid coupon: Please enter your email address…"` and returns). The component, however, sets local `appliedCoupon` state and shows "Coupon Applied: {code}" UI regardless of whether the store accepted it — this is the disconnect.

2. **Checkout page coupon input** (`ApplyCoupon.tsx`, rendered in the right-pane of checkout): triggered by the user clicking the **"Apply"** button on the coupon input. Auto-fills the input value from `sessionStorage` via `getAttribution()` (which reads the `coupon` key from sessionStorage — populated by Coach's external attribution script, NOT by the dealer landing page's `useEffect` directly). Apply still requires the user click.

There is also a parallel auto-fill mechanism: the checkout `ApplyCoupon` component's `useEffect` reads `sessionStorage.getItem('coupon')`. So the dealer-landing → checkout handoff currently depends on Coach's attribution script having written the `coupon` key into sessionStorage during the QR landing, not on the Zustand store (which, per #1 above, will reject the coupon at landing time anyway because email/zip are missing).

### d. Client-side vs server-side

- `validateCoupon` runs **client-side only**, in two places: inside the React component (`ApplyCoupon.tsx:48`) and inside the Zustand store (`useCheckoutStore.ts:216`). Both invocations use the same client-side `couponUtils.ts` function and the same in-memory/persisted `checkoutData`.
- The only server-side step is `GET /api/get-coupon-by-code` (full source in Section 1.f above). It is a thin Woo proxy: takes `?code=…`, forwards to WooCommerce REST `/coupons?code=…` with consumer key/secret, returns the raw Woo coupon array. **It performs no validation** — no email check, no address check, no expiry check, no signature, no auth header, no rate-limit. Anyone with the code can fetch the coupon object including its `meta_data` (allowed emails, expiry, etc.).
- There is no other server-side coupon endpoint (`find src/app/api -type f | grep -i coupon` returns only `get-coupon-by-code/route.ts`). No httpOnly cookie is read or written anywhere in the current flow.
